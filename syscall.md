syscall list got by these steps:
 
1. ```shell
    cargo build --release --no-default-features --features from_mock --target=x86_64-unknown-linux-musl
    ```
2. Using IDA64 to produce assembly codes (free version only supports X86)
3. Search all syscall related codes
4. Verify syscall usage

p.s.

1. NR (call number)
2. x86_64 NR in hex for searching faster in assembly codes generated by IDA64
3. MIPS NR in dec for searching faster in omo syscall list file
4. Some NR passed by edi register for further syscall function invoking (edi is the first arg in UNIX V AMD64 ABI)
5. commit f58f25a73691f4460e3b3842816abda8bdfbceae
6. IDAfree-8.0
